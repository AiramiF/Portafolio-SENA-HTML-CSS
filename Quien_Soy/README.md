# Página Web Personal

Este proyecto consiste en el desarrollo de una página web personal utilizando **HTML y CSS**.  
La página tiene como objetivo presentar información básica sobre mí, mis intereses en el área de tecnología y algunos proyectos de ejemplo.

## Descripción

La página web está diseñada como un **portafolio personal simple**, donde se muestra:

- Información personal
- Una breve descripción sobre mi perfil
- Algunos proyectos de ejemplo
- Una navegación sencilla dentro de la misma página

El diseño es limpio y organizado, pensado para mostrar contenido de manera clara.

## Tecnologías utilizadas

Las tecnologías utilizadas para el desarrollo de este proyecto son:

- **HTML5** para la estructura de la página.
- **CSS3** para el diseño y la apariencia visual.

No se utilizaron frameworks ni librerías externas, todo fue realizado con **HTML y CSS básico**.

## Estructura de la página

La página está dividida en las siguientes secciones:

### Menú de navegación
Ubicado en la parte superior de la página. Permite navegar entre las secciones:
- Inicio
- ¿Quién soy?
- Proyectos

### Sección "¿Quién soy?"
En esta sección se presenta una descripción personal junto con una imagen.

### Sección "Sobre mí"
Contiene información adicional sobre mi formación, experiencia y objetivos profesionales.

### Sección "Proyectos"
Se muestran diferentes tarjetas con ejemplos de proyectos que incluyen:
- Imagen
- Nombre del proyecto
- Descripción breve
- Botón para ver el proyecto

### Footer
En la parte inferior de la página se encuentra el pie de página con el mensaje de copyright.
